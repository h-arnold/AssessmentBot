# ⚠️ v0.7.8

This release is intended to be the last bug-fix release before v1.0.0, which is now being developed on the `react/Frontend` branch. v0.7.8 focuses on fixing a few bugs discovered after v0.7.7 to keep everything running smoothly until v1 is ready.

## ✨ Fixes

- Slides parsing now matches task definitions by task title rather than depending on slide `pageId` parity, fixing cases where student submissions were missed because copied or reordered slides no longer lined up with the reference deck.
- Google Drive document handling is more tolerant of real-world input: reference and template fields now accept more Drive URL shapes, normalise them to file IDs consistently, and reject invalid inputs with clearer errors.
- Class information migration and validation are more reliable, particularly around persisted `YearGroup` values, which reduces upgrade issues and keeps definition matching in step with the selected class.
- Spreadsheet assessment logic and feedback generation were tightened up, with broader unit-test coverage around assessor behaviour, feedback output, and document normalisation paths.

- [src/AdminSheet/GoogleDriveManager/DriveManager.js](src/AdminSheet/GoogleDriveManager/DriveManager.js) and [src/AdminSheet/Utils/Validate.js](src/AdminSheet/Utils/Validate.js) — improved URL validation, query-string handling, and Drive file ID normalisation.
- [src/AdminSheet/ConfigurationManager/ConfigurationManagerClass.js](src/AdminSheet/ConfigurationManager/ConfigurationManagerClass.js), [src/AdminSheet/ConfigurationManager/validators.js](src/AdminSheet/ConfigurationManager/validators.js), and [src/AdminSheet/GoogleClassroom/globals.js](src/AdminSheet/GoogleClassroom/globals.js) — fixed class info migration and validation paths so persisted class metadata, including `YearGroup`, remains reliable after upgrades.
- [src/AdminSheet/Assessors/SheetsAssessor.js](src/AdminSheet/Assessors/SheetsAssessor.js) and [src/AdminSheet/FeedbackPopulators/SheetsFeedback.js](src/AdminSheet/FeedbackPopulators/SheetsFeedback.js) — improved spreadsheet assessment and feedback behaviour.
- [tests/parsers/parserMatchingAndDocumentIds.test.js](tests/parsers/parserMatchingAndDocumentIds.test.js), [tests/googleDriveManager/driveManager.normaliseToFileId.test.js](tests/googleDriveManager/driveManager.normaliseToFileId.test.js), [tests/configurationManager/classInfoMigration.test.js](tests/configurationManager/classInfoMigration.test.js), [tests/configurationManager/validateClassInfo.test.js](tests/configurationManager/validateClassInfo.test.js), [tests/assessors/sheetsAssessor.test.js](tests/assessors/sheetsAssessor.test.js), and [tests/feedbackPopulators/sheetsFeedback.test.js](tests/feedbackPopulators/sheetsFeedback.test.js) — added regression coverage for the patched paths.

## ⬆️ Updating to v0.7.8

1. Open the Admin Sheet and choose Assessment Bot → Settings → Update Version.
2. Select v0.7.8 from the dropdown and follow the prompts to reinstall the latest script bundle.
3. No additional data migration is required beyond the v0.7.6 steps.
